syntax reference ▾

Elements

<element> a, b, c — declare elements (in the root container)
<X> a, b, c — declare a, b, c as members of X (X is also an element)
Every element is in <element> by default. Any element can be a container.

Relations

rel name/arity — declare a relation (accepts any element)
rel name/arity : X, Y — arg 1 must be in X, arg 2 in Y
rel name/arity : equivalence — + symmetry, transitivity, reflexivity, congruence
rel name/arity : X, Y, transitive — membership + properties can mix

Rules

rule name: premise1, premise2 |- conclusion1, conclusion2
Variables are prefixed with ? — e.g. R(?x, ?y) |- R(?y, ?x)

Facts

relation(term1, term2, ...) — ground atoms (all terms must be declared constants)

Commands

derive — compute closure and display all derived facts
derive N — limit to N rounds (default: 100)
prove R(a, b) — compute closure and show proof tree for a goal

Comments

Lines starting with # are ignored.
Output
Write a program and press Run to compute closure.
